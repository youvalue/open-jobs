## English

**OpenJobs** is a completely free, zero-infrastructure hiring platform. **All features are fully open to everyone** — post jobs, submit resumes, find talent, all at no cost.

### Why is it free?

OpenJobs runs entirely on GitHub's free services. No servers, no databases, no subscriptions.

| | Traditional Platforms | OpenJobs |
|---|---|---|
| **Monthly Cost** | $200+/mo | **$0** |
| **Infrastructure** | Servers, DBs, CDN | **GitHub Pages (free)** |
| **Data Storage** | Private database | **GitHub Issues (open, exportable)** |
| **Authentication** | Email/password signup | **GitHub OAuth (no new account needed)** |
| **Moderation** | Black box algorithms | **GitHub Actions + open-source rules** |

### What's free?

**For job seekers** — Browse all listings, submit your resume, get discovered — all free.

**For employers** — Post unlimited jobs, browse candidates, send interview invitations — all free.

**Your data, your control** — Every resume and job is a GitHub Issue. Fork the repo, take your data with you. No lock-in.

**No extra signup** — One-click GitHub login. No new passwords, no lengthy forms.

**Global from day one** — 7 languages, 26 job roles, multi-country city data.

### Tech Stack

Pure static SPA. Zero build step. Push to `main` → GitHub Pages deploys automatically.

- **Hosting**: GitHub Pages (free)
- **Data**: GitHub Issues (free, unlimited)
- **Automation**: GitHub Actions for spam detection and email
- **Auth**: GitHub OAuth (free)

**No hidden fees. No paywalls. Ever.**
